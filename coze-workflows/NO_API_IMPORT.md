# 🎯 无需API Key的导入方案

如果找不到API Key，您可以直接在Coze平台手动创建工作流，只需**复制粘贴**即可！

---

## 📋 准备工作（2分钟）

1. **登录Coze平台**
   - 访问：https://www.coze.cn
   - 创建一个Bot（如果还没有）

2. **打开两个文档**
   - 📄 `PROMPTS.md` - 包含所有Prompt
   - 📄 `MANUAL_CONFIG.md` - 包含详细配置

---

## 🚀 快速创建工作流（每个约5分钟）

### 示例：创建"文献检索式生成"工作流

#### 第1步：创建工作流
1. 进入Bot编辑页面
2. 点击左侧「工作流」→「创建工作流」
3. 名称输入：`search_query_generation`

#### 第2步：添加开始节点（已有，配置输入）

在开始节点添加2个输入变量：

**变量1：**
```
名称：research_question
类型：String
描述：研究问题
必填：✅
```

**变量2：**
```
名称：search_strategy
类型：String
描述：检索策略
默认值：precise
必填：❌
```

#### 第3步：添加LLM节点

1. 点击「+」添加「LLM」节点
2. 配置：
   ```
   模型：选择扣子内置模型（如豆包）
   Temperature：0.3
   ```

3. **复制Prompt**：
   - 打开 `PROMPTS.md`
   - 找到"文献检索式生成"部分
   - 复制 System Prompt，粘贴到节点配置
   - 复制 User Prompt，粘贴到节点配置

4. 连接节点：
   ```
   开始 → LLM
   ```

#### 第4步：添加结束节点

1. 添加「结束」节点
2. 配置输出变量：
   ```
   名称：result
   类型：Object
   值：{{LLM节点输出}}
   ```

3. 连接：
   ```
   开始 → LLM → 结束
   ```

#### 第5步：保存并测试

1. 点击「保存」
2. 点击「测试」
3. 输入测试数据：
   ```json
   {
     "research_question": "PGT对IVF结局的影响"
   }
   ```
4. 查看输出结果

## 📝 所有工作流创建清单

按照相同方法创建其他4个工作流：

### 1. 文献AI初筛
- 打开 `PROMPTS.md` → "文献AI初筛"
- 复制Prompt到LLM节点
- 输入：research_question, articles

### 2. 文献数据提取
- 打开 `PROMPTS.md` → "文献数据提取"
- 复制Prompt到LLM节点
- 输入：literature_content

### 3. 文献质量评估
- 打开 `PROMPTS.md` → "文献质量评估"
- 需要创建2个LLM节点（RCT用RoB 2.0，观察性研究用NOS）
- 输入：literature_content, study_type

### 4. Meta分析计算
- 这个工作流主要是代码节点
- 打开 `MANUAL_CONFIG.md` → "Meta分析计算"
- 复制JavaScript代码到代码节点
- 输入：studies（研究数据数组）

## 🎨 配置Bot人设（3分钟）

### 方法：

1. **进入Bot编辑页面**
2. **找到「人设与回复逻辑」**
3. **复制以下内容：**

```
你是一位专业的循证医学研究助手，精通Meta分析方法论。

## 核心能力
1. 文献检索：根据研究问题生成PICO/PECO框架检索式
2. 文献筛选：AI辅助判断文献相关性
3. 数据提取：从文献中提取研究数据
4. 质量评估：使用Cochrane RoB 2.0等量表评估
5. Meta分析：固定/随机效应模型分析

## 工作流程
- 用户提出研究问题 → 引导生成检索式
- 用户上传文献 → 自动分类和筛选
- 用户请求数据提取 → 引导提取关键信息
- 用户请求分析 → 执行Meta分析并展示结果

## 沟通风格
- 专业但易懂
- 主动引导用户完成步骤
- 提供详细解释和建议
```

4. **点击「保存」**

## 🆘 遇到问题？

1. **找不到节点类型？**
   - 点击工作流编辑器中的「+」按钮
   - 查看所有可用节点类型

2. **Prompt太长？**
   - 分段复制粘贴
   - 或使用浏览器的剪贴板历史功能

3. **节点连接错误？**
   - 拖拽节点重新连接
   - 删除节点重新添加

4. **测试失败？**
   - 检查输入参数格式
   - 查看节点日志定位问题

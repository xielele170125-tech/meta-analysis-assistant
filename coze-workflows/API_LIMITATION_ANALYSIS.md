4. **Template API** - 不存在
5. **Import/Export API** - 不存在

---

## 🤔 为什么Coze不支持工作流API？

### 1. 安全性考虑
工作流可能包含复杂的逻辑和敏感操作，通过UI创建可以：
- 让用户明确了解每个节点的功能
- 防止恶意批量创建工作流
- 控制资源消耗

### 2. 复杂性控制
工作流涉及：
- 多个节点的连接关系
- 变量的传递和引用
- 条件分支和循环
- 错误处理逻辑

通过UI创建更直观，减少出错概率。

### 3. 平台策略
Coze希望：
- 用户通过UI界面熟悉产品功能
- 控制工作流的创建质量
- 避免滥用API批量创建

### 4. 产品定位
Coze主要定位为：
- 低代码/无代码平台
- 面向非技术用户
- 通过可视化界面操作

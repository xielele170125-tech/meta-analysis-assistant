# ⚡ 超简化版 - 5步创建第一个工作流

## 目标
创建一个"文献检索式生成"工作流

---

## 第1步：创建Bot（2分钟）
1. 打开 https://www.coze.cn
2. 点击「创建Bot」
3. 输入名称：`Meta分析助手`
4. 点击「确定」

---

## 第2步：创建工作流（1分钟）
1. 左侧菜单 → 工作流 → 创建
2. 输入名称：`search_query_generation`
3. 点击「确定」

---

## 第3步：配置输入（2分钟）

### 添加变量1：
```
名称：research_question
类型：String
必填：✓
```

### 添加变量2：
```
名称：search_strategy
类型：String
默认：precise
必填：不勾选
```

---

## 第4步：添加LLM节点（3分钟）

### 4.1 添加节点
- 点击「+ 添加节点」
- 选择「LLM」

### 4.2 复制System Prompt
复制这段内容，粘贴到System Prompt框：

```
你是一位循证医学专家，精通多种研究问题框架。

## 常见研究框架

### 1. PICO（干预性研究）
- P (Population): 研究人群
- I (Intervention): 干预措施
- C (Comparison): 对照措施
- O (Outcome): 结局指标

## 任务
1. 首先判断研究问题属于哪种类型
2. 提取相应的要素
3. 如果问题宽泛，主动推断和扩展可能的要素
4. 识别关键概念和同义词
```

### 4.3 复制User Prompt
复制这段内容，粘贴到User Prompt框：

```
研究问题：{{start.research_question}}

请分析研究类型，提取关键要素。
```

### 4.4 连接节点
拖拽「开始」节点到「LLM」节点

---

## 第5步：测试（1分钟）

### 5.1 保存
点击右上角「保存」

### 5.2 测试
1. 点击「测试」
2. 输入：
   ```
   PGT对IVF结局的影响
   ```
3. 点击「运行」

### 5.3 查看结果
如果看到输出了JSON格式的分析结果，就成功了！

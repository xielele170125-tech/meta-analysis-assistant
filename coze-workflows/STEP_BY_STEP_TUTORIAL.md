## 第一步：创建Bot（2分钟）

### 1.1 打开Coze平台

**操作：**
1. 打开浏览器
2. 访问：https://www.coze.cn
3. 点击右上角「登录」（用抖音/飞书/微信登录）

**界面：**
```

│  Coze Logo        [登录] [注册]        │
```

### 1.2 创建新Bot

**操作：**
1. 登录后，点击页面上的「创建Bot」按钮（大按钮，很显眼）
2. 在弹出的对话框中填写：
   ```
   Bot名称：Meta分析助手
   Bot简介：专业的文献Meta分析AI助手
   ```
3. 点击「确定」或「创建」

**操作：**
1. 在左侧菜单栏中，找到「工作流」选项
2. 点击旁边的「+」号或「创建」按钮

### 4.3 复制并粘贴Prompt

这是最关键的一步！请仔细操作：

#### 操作A：复制System Prompt

**从这里复制：**
```
你是一位循证医学专家，精通多种研究问题框架。

## 常见研究框架

### 1. PICO（干预性研究）
- P (Population): 研究人群
- I (Intervention): 干预措施
- C (Comparison): 对照措施
- O (Outcome): 结局指标

### 2. PECO（暴露研究/观察性研究）
- P (Population): 研究人群
- E (Exposure): 暴露因素
- C (Comparison): 对照/非暴露
- O (Outcome): 结局指标

### 3. SPIDER（定性/混合研究）
- S (Sample): 样本
- PI (Phenomenon of Interest): 感兴趣的现象
- D (Design): 研究设计
- E (Evaluation): 评估方法
- R (Research type): 研究类型

## 任务
1. 首先判断研究问题属于哪种类型
2. 提取相应的要素
3. 如果问题宽泛，**主动推断和扩展**可能的要素
4. 识别关键概念和同义词
5. 提供相关的MeSH主题词

【重要】即使问题不完整，也要尽可能推断和扩展，而不是返回空值。
```

**粘贴到：**
- Coze平台右侧面板的「System Prompt」文本框中

#### 操作B：复制User Prompt

**从这里复制：**
```
研究问题：{{start.research_question}}

请分析研究类型，提取关键要素，并尽可能扩展相关概念。如果问题宽泛，请主动推断可能的PICO/PECO要素。
```

**粘贴到：**
- Coze平台右侧面板的「User Prompt」文本框中

**注意：** `{{start.research_question}}` 这个是变量引用，不要修改！

## 🆘 常见问题

### Q1: 找不到节点类型？
**A:** 点击画布上的「+」号，会显示所有可用节点类型

### Q2: 节点连接不上？
**A:** 拖拽节点的输出端（右侧小圆点）到下一个节点的输入端（左侧小圆点）

### Q3: 测试报错？
**A:** 检查：
- 输入变量名称是否正确
- 变量引用格式是否正确 `{{节点名.变量名}}`
- Prompt是否完整粘贴

### Q4: 不知道变量如何引用？
**A:** 
- 开始节点的变量：`{{start.变量名}}`
- LLM节点的输出：`{{llm节点名.output}}`
- 点击变量输入框，通常会显示可选变量列表

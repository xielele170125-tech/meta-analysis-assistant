# 🔍 Coze API Key 获取指南（2024最新版）

## 问题：找不到右上角头像？

别担心，Coze平台界面可能有更新。请按以下方法查找：

---

## 方法1：从个人中心获取

### 步骤：

1. **登录Coze平台**
   - 访问：https://www.coze.cn
   - 使用抖音/飞书/微信登录

2. **进入个人中心**
   - 登录后，点击页面左下角的**个人头像**或**用户名**
   - 或者访问：https://www.coze.cn/user/me

3. **找到API管理**
   - 在左侧菜单栏中找到「API管理」或「开发者设置」
   - 如果没有，尝试点击「设置」→「API配置」

---

## 方法2：直接访问API管理页面

登录后直接访问以下链接：
```
https://www.coze.cn/api/manage
或
https://www.coze.cn/user/settings/api
```

---

## 方法3：从Bot编辑页面获取

1. **创建或进入一个Bot**
   - 点击「创建Bot」
   - 或进入已有的Bot编辑页面

2. **在Bot配置页面**
   - 查找「API」或「集成」标签
   - 可能在这里直接创建API Key

---

## 方法5：新用户特殊流程

如果您是新注册用户，可能需要：

1. **完成新手引导**
   - 创建第一个Bot
   - 设置Bot基本信息

2. **开通API权限**

   - 在Bot配置页面查找API相关选项

## 🆘 如果仍然找不到

### 方案A：使用Bot内的API功能

1. **创建一个测试Bot**
   ```
   名称：Meta分析测试Bot
   简介：测试用
   ```

2. **在Bot配置页面查找**
   - 发布设置
   - API集成
   - Webhook配置

3. **从这里获取API Key**

### 方案B：使用手动创建方式

如果实在找不到API Key，可以：

1. **参考 MANUAL_CONFIG.md**
   - 手动在工作流编辑器中创建节点
   - 复制Prompt和代码

2. **不需要API导入**
   - 在Coze平台直接编辑工作流
   - 逐个节点配置

- Coze帮助中心：https://www.coze.cn/help

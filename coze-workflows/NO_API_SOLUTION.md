## 📖 使用步骤

### 第1步：打开配置工具
访问 http://localhost:5000/coze-config.html

### 第2步：选择工作流
点击顶部的标签，选择要创建的工作流

### 第3步：复制Prompt
- 点击「复制 System Prompt」按钮
- 点击「复制 User Prompt」按钮

### 第4步：在Coze平台粘贴
1. 打开 https://www.coze.cn
2. 创建或编辑工作流
3. 添加LLM节点
4. 粘贴复制的Prompt

### 第5步：连接并测试
按照界面提示完成工作流配置

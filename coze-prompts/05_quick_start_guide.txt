# 快速操作指南 - 5分钟创建Coze Bot

## 📋 操作步骤

### 第一步：创建Bot（2分钟）

1. 打开浏览器，访问 **https://www.coze.cn**
2. 登录账号（抖音/飞书/手机号）
3. 点击右上角「创建Bot」
4. 填写基本信息：
   ```
   Bot名称：文献Meta分析助手
   Bot描述：专业的医学文献Meta分析AI助手
   ```
5. 点击「确定」创建

---

### 第二步：配置人设（2分钟）

1. 在Bot编辑页面，找到「人设与回复逻辑」区域
2. 复制以下内容，粘贴进去：

```
# 角色
你是专业的循证医学研究助手，精通Meta分析方法论。

## 核心能力
1. 检索式生成：根据研究问题生成PubMed/EMBASE/Cochrane检索式
2. 文献初筛：判断文献是否与研究问题相关
3. 数据提取：提取样本量、效应量等Meta分析数据
4. 质量评估：使用Cochrane RoB 2.0、NOS量表评估文献质量
5. 方法咨询：回答Meta分析方法问题

## 工作流程
识别用户需求 → 执行对应功能 → 提供结构化输出

## 检索式生成规则
- 使用PICO框架：人群P + 干预I + 对照C + 结局O
- 提取关键词和MeSH词
- 提供三种策略：精确(100-2000篇)、敏感(1000-5000篇)、限定(50-500篇)

## 文献筛选原则
- 宽松纳入：可能相关就标记"纳入"
- 严格排除：明确不相关才标记"排除"
- 提供判断理由和置信度

## 数据提取要点
- 研究名称：第一作者(年份)
- 样本量：治疗组/对照组
- 数据类型：连续型/二分类
- 结局指标：标准化命名

## 质量评估标准
- RCT研究：Cochrane RoB 2.0（5个域）
- 观察性研究：Newcastle-Ottawa量表（选择/可比性/结局）

## 沟通风格
专业但易懂，主动引导，提供具体建议

## 引导转化
当用户需要森林图、批量处理、Excel导出时：
"我可以帮你完成基础分析！如需完整功能（可视化图表、批量处理、导出报告），推荐使用专业版Web应用。"
```

3. 点击「保存」

---

### 第三步：选择模型（30秒）

1. 在Bot编辑页面，找到「模型」选择区域
2. 选择推荐模型：
   - **豆包Pro**（推荐，中文效果好）
   - **DeepSeek V3**（推理能力强）
   - **Kimi**（长文本处理）

---

### 第四步：测试Bot（1分钟）

1. 点击右上角「预览」按钮
2. 在右侧对话框输入测试问题：

**测试1 - 检索式生成**：
```
我想研究二甲双胍对2型糖尿病患者心血管事件的影响，请帮我生成检索式
```

**测试2 - 文献筛选**：
```
请判断这篇文献是否相关：
标题：Effect of metformin on cardiovascular outcomes in type 2 diabetes
摘要：This randomized controlled trial investigated the effect of metformin on cardiovascular events...
```

**测试3 - 质量评估**：
```
请用Cochrane RoB 2.0评估这篇RCT研究的质量：
[粘贴文献内容]
```

3. 观察回复是否符合预期
4. 如需调整，修改「人设与回复逻辑」中的内容

---

### 第五步：发布Bot（1分钟）

1. 确认测试通过后，点击右上角「发布」
2. 选择「发布到Bot商店」
3. 填写商店信息：

```
一句话介绍：专业的医学文献Meta分析AI助手，支持检索式生成、文献筛选、数据提取、质量评估

详细介绍：
## 功能特色

### 🔍 智能检索式生成
- 支持PICO/PECO/SPIDER研究框架
- 自动识别研究要素和MeSH主题词
- 生成PubMed、EMBASE、Cochrane检索式

### 📋 AI文献初筛
- 自动判断文献相关性
- 提供判断理由和置信度

### 📊 数据提取
- 自动识别研究设计类型
- 提取样本量、效应量、结局指标

### ✅ 质量评估
- Cochrane RoB 2.0（RCT研究）
- Newcastle-Ottawa量表（观察性研究）

## 适用人群
医学研究者、公共卫生学者、临床医生、医学研究生

## 使用场景
系统评价、Meta分析研究、循证决策、科研写作
```

4. 上传图标（可用AI生成或上传图片）
5. 点击「提交审核」

---

## ✅ 完成！

你的Bot已经提交审核，通常1-3个工作日会通过审核并上架到Coze商店。

---

## 🔧 进阶配置（可选）

如果你想添加更强大的功能，可以：

### 1. 创建工作流

- 工作流可以让Bot执行更复杂的任务
- 参考 `COZE_BOT_CONFIG.md` 中的工作流配置
- 每个工作流对应一个具体功能

### 2. 添加插件

- 可以添加HTTP插件调用外部API
- 可以连接数据库存储数据
- 可以集成第三方服务

### 3. 添加知识库

- 上传PDF文档作为知识库
- 让Bot基于上传的文档回答问题
- 适合添加方法学指南等参考资料

## 📞 遇到问题？

### 常见问题

**Q：人设字数超限怎么办？**
A：使用精简版本，只保留核心内容

**Q：找不到"大模型节点"？**
A：不创建工作流，直接用对话模式即可

**Q：回复质量不好？**
A：在Prompt中添加更多示例和要求

**Q：如何变现？**
A：Coze支持扣子币付费，或引导到你的Web应用

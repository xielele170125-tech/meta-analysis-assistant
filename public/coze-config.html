🎯 Coze工作流配置工具

一键复制所有Prompt，快速创建工作流

📋 创建第一个工作流的步骤

1
创建Bot
访问 https://www.coze.cn → 创建Bot → 名称填写"Meta分析助手"
2
创建工作流
左侧菜单 → 工作流 → 创建 → 名称填写"search_query_generation"
3
配置输入变量
在开始节点添加两个变量：
- research_question (String, 必填)
- search_strategy (String, 默认: precise)
4
添加LLM节点
点击"+ 添加节点" → 选择"LLM" → 点击下方对应工作流标签，复制Prompt粘贴即可
5
连接并测试
连接开始节点到LLM节点 → 保存 → 测试运行

点击上方的工作流标签，可以直接复制对应工作流的所有Prompt配置。
建议先从"检索式生成"工作流开始，熟悉流程后再创建其他工作流。

节点：提取数据
System Prompt (点击复制后需要在Coze中查看完整内容):

Meta分析计算

执行固定/随机效应模型Meta分析，计算合并效应量和异质性

节点：解读分析结果
System Prompt:
你是一位专业的循证医学专家，请解读Meta分析结果。 ## 解读要点 ### 1. 效应量解读 根据效应量类型进行解读： - RR (相对风险)：RR<1表示干预降低风险，RR>1表示增加风险 - OR (比值比)：OR<1表示干预降低几率，OR>1表示增加几率 - HR (风险比)：HR<1表示干预降低风险率 - SMD (标准化均数差)：SMD<0.2小效应，0.2-0.8中效应，>0.8大效应 - MD (均数差)：直接解释差异大小 ### 2. 异质性解读 - I² < 25%：低异质性 - I² 25-50%：中等异质性 - I² 50-75%：较高异质性 - I² > 75%：高异质性 ### 3. 统计学显著性 - P < 0.05：统计学显著 - P ≥ 0.05：无统计学显著性 ### 4. 临床意义 综合考虑效应量大小、置信区间、研究质量等因素 ## 输出格式 请用通俗易懂的语言撰写解读报告，包含以下部分： ```json { "main_findings": "主要发现（2-3句话）", "effect_interpretation": "效应量解读", "heterogeneity_interpretation": "异质性解读", "clinical_significance": "临床意义", "limitations": ["局限性1", "局限性2"], "conclusions": "结论" } ```

# 收款码图片说明

## 如何替换收款码

### 方法一：直接替换文件

1. **获取收款码图片**
   - 微信：打开微信 → 我 → 服务 → 收付款 → 二维码收款 → 保存图片
   - 支付宝：打开支付宝 → 首页 → 收钱 → 保存图片

2. **重命名图片**
   - 微信收款码 → 重命名为 `wechat-qr.png`
   - 支付宝收款码 → 重命名为 `alipay-qr.png`

3. **放置图片**
   将图片放到 `public/payment/` 目录下，替换现有的占位图

### 方法二：截图方式

如果无法保存图片，可以截图：
1. 打开收款码页面
2. 截图保存
3. 裁剪为正方形（推荐 300x300 或 400x400 像素）
4. 按上述方式重命名并放置

## 注意事项

- 图片格式：PNG 或 JPG 均可
- 推荐尺寸：300x300 像素
- 确保收款码清晰可扫描
- 不要修改文件名，否则需要同步修改代码

## 验证

替换后，启动项目并访问支付弹窗，确认收款码正常显示。

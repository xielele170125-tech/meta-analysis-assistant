# GitHub 网页上传文件指南

## 你的仓库地址
https://github.com/xielele170125-tech/meta-analysis-assistant

---

## 第1步：创建仓库（如果还没创建）

1. 访问 https://github.com/new
2. Repository name: `meta-analysis-assistant`
3. 选择 Public
4. **不要勾选**任何选项
5. 点击 "Create repository"

---

## 第2步：上传文件

### 方式A：网页上传（最简单）

1. 打开你的仓库：https://github.com/xielele170125-tech/meta-analysis-assistant
2. 点击 **"uploading an existing file"**
3. 把文件拖进去
4. 点击 **"Commit changes"**

### 方式B：GitHub Desktop上传

1. 打开GitHub Desktop
2. File → Add Local Repository → 选择项目文件夹
3. 点击 **"Publish repository"**

# 项目文件创建指南

## 📁 第一步：在电脑上创建文件夹结构

在你的电脑上创建以下文件夹：

```
meta-analysis-assistant/
├── src/
│   ├── app/
│   │   ├── api/
│   │   │   ├── analysis/
│   │   │   ├── literature/
│   │   │   ├── export/
│   │   │   └── ...
│   │   └── ...
│   ├── components/
│   ├── lib/
│   └── storage/
├── public/
├── scripts/
└── (根目录配置文件)
```

如果你有 **Git** 和 **Node.js**，可以直接在命令行执行：

```bash

mkdir meta-analysis-assistant
cd meta-analysis-assistant

# 初始化Git
git init

# 创建远程仓库连接

# 拉取我帮你准备的模板
```
